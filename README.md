# Ela Compila

Blog técnico criado para compartilhar tutoriais rápidos, aprendizados práticos e conteúdos sobre Docker, backend, APIs e automação com Python.

## Acesse o blog

**Produção:**  
[ela-compila.vercel.app/blog](https://ela-compila.vercel.app/blog)

## Sobre o projeto

O **Ela Compila** foi desenvolvido como parte do meu portfólio para reunir conteúdos técnicos de forma organizada, visualmente agradável e fácil de compartilhar.

A proposta do projeto é demonstrar conhecimento em:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Estruturação de interfaces
- Organização de conteúdo técnico
- Publicação e deploy com Vercel

## Objetivo

Este repositório faz parte do meu portfólio profissional e foi criado para:

- publicar artigos técnicos
- consolidar aprendizados de projetos reais
- compartilhar conteúdo de forma prática no LinkedIn
- demonstrar habilidades em frontend e organização de produto

## Tecnologias utilizadas

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Vercel**
